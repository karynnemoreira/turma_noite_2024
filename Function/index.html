Função em JS é um conjunto de instruções que realizam uma tarefa.

Parâmetros(): É declarada entre parênteses, é um valor que pode ser alterado durante a execução.

Declarando funções (function) função - nome para a função () parâmetro da função { tarefa, o que quero que essa função faça } A função precisa ser chamada para ser executada (invocada/ativada).
